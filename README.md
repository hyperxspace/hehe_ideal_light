# hehe ideal light

## 对理想X星说HEHE,献给爱技术的你

### 快速开始



####  克隆项目

```shell
git clone https://github.com/hyperxspace/hehe_ideal_light.git
npm install # or cnpm install`
```

#### 运行项目

```shell :
# 连接手机
# 打开XX 进入到题目界面
node app.js
# 切换题目后按回车进入下一题的识别
# enjoy it :)
```

### 环境要求

- nodejs
- 确保adb在环境变量中
